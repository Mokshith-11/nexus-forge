# Nexus Forge — Get Found on Google (SEO & Visibility Plan)

**Goal:** When people search "website designing", "AI automation agency", "WhatsApp automation", etc., Nexus Forge shows up — ideally near the top.

**Honest reality:** Broad terms ("automation", "website designing") are won over months, or instantly via paid ads. Specific/local terms ("AI automation agency in [city]", "website designing in [city]", "WhatsApp chatbot for clinics") are winnable much faster. This plan attacks both.

---

## Two lanes running in parallel

### 🐢 Lane A — Organic (free, compounding, weeks→months)
### ⚡ Lane B — Google Ads (paid, instant top placement)

---

## PHASE 0 — Foundations ✅ DONE
- Site live, fast, HTTPS, mobile-friendly
- Indexed on Google + sitemap submitted (Search Console)
- Google Business Profile live & verified (huge for "near me" searches)
- Analytics + GA4 event tracking

## PHASE 1 — On-Page SEO (Claude does now · week 1)
Make Google understand exactly what you sell and who for.
- **Keyword-optimized titles & meta** on the homepage and each key section (target: "AI Automation Agency", "Website Designing & Development", "WhatsApp Automation", "AI Chatbots").
- **Headings (H1/H2)** rewritten to include money keywords naturally.
- **Service structured data (JSON-LD)** for each service → eligible for rich results.
- **Dedicated landing pages** for the big keywords, e.g. `/website-design`, `/ai-automation`, `/whatsapp-automation` — each optimized for one keyword + a CTA. This is what actually ranks for specific terms.
- **Local pages** (if targeting a city): `/[city]/website-design` style — these win "website designing in [city]" fast.
- Internal links + descriptive image alt text.

## PHASE 2 — Content Engine (weeks 1–6, ongoing)
Long-tail keywords are where a new site wins first. Publish a blog (you sell content — demo it on yourself):
- "How much does website designing cost in India (2026)"
- "Best AI automation tools for small business"
- "WhatsApp chatbot for clinics: complete guide"
- "How AI automation saves agencies 20 hours a week"
- 2 posts/month, each targeting one keyword people actually search.
- Claude can build the blog + write the first posts.

## PHASE 3 — Authority & Local Ranking (ongoing)
Google ranks sites it trusts. Build trust signals:
- **Get GBP reviews** — ask every happy client. 5+ reviews dramatically boosts "near me" ranking.
- **Directories/backlinks** — list Nexus Forge on Clutch, GoodFirms, Sortlist, Justdial, Sulekha, LinkedIn. Free links = ranking power.
- **Social presence** — post on LinkedIn/Instagram, link back to the site.
- **Real case studies** on the site (after client #1) — unique content Google loves.

## LANE B — Google Ads (instant, parallel, paid)
For "at the front today":
- Campaign type: **Search Ads**.
- Target keywords: `website designing agency`, `ai automation agency`, `whatsapp automation service`, `[city] web development`.
- Budget: start ₹300–500/day, adjust by results.
- Send clicks to the matching landing page (from Phase 1) — not the homepage.
- Claude writes the ad copy + keyword list; you set it up + set budget.

| When | What you can expect |
